XML Splitter

Upload a single XML file containing multiple objects under one root tag, and download a ZIP with one XML per object.

The XML Splitter takes an XML file containing multiple objects nested under one root (e.g. <Order>, <Product>, <Customer>) and splits them into individual XML files. Each object becomes its own XML file, and all files are delivered as a ZIP archive to your default downloads folder.
